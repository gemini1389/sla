# 2. Основи технології SLA

Що охоплює цей розділ:

- **як працює фотополімерний друк** — процес друку простими словами
- **SLA, MSLA, DLP** — у чому різниця між варіантами технології
- **основні терміни та поняття** — словник для подальшого читання

---

## 2.1. Як працює фотополімерний друк (простими словами)

Фотополімерний друк — це процес перетворення рідкої смоли на твердий пластик за допомогою світла. Уявіть собі, що ви малюєте картину, але замість фарби використовуєте рідку смолу, яка миттєво затвердіває під ультрафіолетовим світлом.

### Основний принцип роботи:

1. **Ванночка зі смолою** — принтер має прозору ванночку (vat), заповнену рідкою фотополімерною смолою. Знизу ванночки знаходиться прозора плівка (FEP або ACF), яка дозволяє світлу проходити крізь неї.

2. **Платформа для друку** — над ванночкою розташована платформа (build plate), яка опускається в смолу. Спочатку вона знаходиться на самому верху.

3. **Процес друку шару**:
    - Платформа опускається в смолу на глибину одного шару (зазвичай 0.025–0.1 мм)
    - Під ванночкою ультрафіолетове світло висвітлює ті ділянки, які мають стати частиною моделі
    - Смола в цих місцях миттєво затвердіває (полімеризується) і прилипає до платформи
    - Платформа трохи піднімається, відриваючи затверділий шар від плівки
    - Процес повторюється для наступного шару

4. **Накопичення шарів** — кожен новий шар прилипає до попереднього, поступово формуючи всю модель знизу вгору.

### Чому це працює?

Фотополімерна смола містить спеціальні молекули (мономери), які під дією ультрафіолетового світла з'єднуються в довгі ланцюги (полімери). Це хімічна реакція, яка відбувається дуже швидко — за секунди. Після полімеризації смола вже не може повернутися в рідкий стан при нормальній температурі.

### Аналогія для кращого розуміння:

Уявіть, що ви будуєте будинок з цементу:
- **Смола** = рідкий цемент
- **Ультрафіолетове світло** = вода, яка робить цемент твердим
- **Шари** = поверхи будинку, які ви будуєте один на одному
- **Плівка FEP** = опалубка, яка дозволяє створювати точні контури кожного шару

## 2.2. SLA, MSLA, DLP — у чому різниця

Всі три технології використовують фотополімеризацію, але відрізняються тим, **як** вони створюють ультрафіолетове світло для висвітлення шару.

### SLA (Stereolithography) — класична технологія

**Як працює:** Лазерний промінь, який рухається по поверхні смоли, "малює" кожен шар по точках.

**Переваги:**
- Найвища точність та якість поверхні
- Можливість друкувати дуже дрібні деталі
- Професійні результати

**Недоліки:**
- Дороге обладнання (зазвичай від $3000+)
- Повільніший друк (лазер малює кожну точку окремо)
- Складніше обслуговування

**Де використовується:** Професійні принтери для медицини, ювелірної справи, точного прототипування.

### MSLA (Masked Stereolithography) — найпопулярніша для домашніх принтерів

**Як працює:** LCD-екран (як у смартфоні) з ультрафіолетовою підсвіткою показує "маску" всього шару одночасно. Вся площа шару висвітлюється за один раз.

**Переваги:**
- Швидкий друк (весь шар одночасно, незалежно від складності)
- Доступна ціна (від $200)
- Просте обслуговування
- Висока якість для більшості завдань

**Недоліки:**
- Трохи нижча точність порівняно з SLA (залежить від роздільної здатності LCD)
- LCD-екран має обмежений термін служби (зазвичай 2000–4000 годин)

**Де використовується:** Більшість настільних SLA-принтерів (Elegoo, Anycubic, Phrozen тощо).

### DLP (Digital Light Processing) — технологія з проектором

**Як працює:** Цифровий проектор висвітлює весь шар одночасно, як MSLA, але використовує мікро-дзеркала (DMD) для формування зображення.

**Переваги:**
- Швидкий друк (весь шар одночасно)
- Висока якість та точність
- Довговічність проектора

**Недоліки:**
- Дорожче за MSLA
- Може бути обмеження по розміру області друку
- Менш поширене в настільних принтерах

**Де використовується:** Професійні принтери середнього класу, деякі настільні моделі.

### Яку технологію обрати?

**Для більшості початківців:** **MSLA** — оптимальний баланс між ціною, якістю та швидкістю.

**Для професійного використання:** **SLA** або **DLP** — якщо потрібна максимальна точність та готовність інвестувати більше.

**Важливо:** Незалежно від технології, всі вони використовують однакові смоли та мають схожі процеси післяобробки. Різниця лише в джерелі світла та швидкості друку.

## 2.3. Основні терміни та поняття

### Обладнання та компоненти

**Build Plate (Платформа для друку)** — металева платформа, до якої прилипає перший шар моделі. Під час друку опускається в смолу.

**Vat (Ванночка)** — контейнер зі смолою, в який опускається платформа. Зазвичай має прозоре дно.

**FEP (Fluorinated Ethylene Propylene)** — прозора плівка на дні ванночки, через яку проходить ультрафіолетове світло. Міцна, але потребує регулярної заміни.

**ACF (Anti-Creep Film)** — альтернатива FEP, новіша технологія. Менше опору при відриві, довший термін служби, але дорожча.

**LCD Screen (LCD-екран)** — у MSLA-принтерах створює "маску" для висвітлення шару. Має обмежений термін служби.

**UV LED Array (Масив UV-світлодіодів)** — джерело ультрафіолетового світла в MSLA-принтерах, розташоване під LCD-екраном.

### Процес друку

**Layer Height (Висота шару)** — товщина кожного шару друку. Типові значення: 0.025 мм (висока якість), 0.05 мм (стандарт), 0.1 мм (швидкий друк). Менша висота = краща якість, але повільніший друк.

**Exposure Time (Час експозиції)** — скільки часу світло висвітлює кожен шар. Занадто мало = недрукування, занадто багато = передрукування та втрата деталей.

**Bottom Layer Count (Кількість нижніх шарів)** — перші кілька шарів друкуються з більшим часом експозиції для кращого прилипання до платформи.

**Bottom Exposure Time (Час експозиції нижніх шарів)** — час висвітлення для перших шарів. Зазвичай в 5–10 разів більший за звичайний.

**Lift Speed (Швидкість підйому)** — як швидко платформа піднімається після висвітлення шару. Занадто швидко = відрив від платформи.

**Retract Speed (Швидкість опускання)** — як швидко платформа опускається для наступного шару.

**Rest Time (Час відпочинку)** — пауза після опускання платформи, щоб смола встигла розтіктися.

### Підтримки та орієнтація

**Supports (Підтримки)** — тонкі структури, які підтримують частини моделі, що нависають. Після друку видаляються.

**Raft (Плита)** — товстий перший шар під моделлю, який полегшує зняття з платформи.

**Island (Острів)** — частина моделі, яка не з'єднана з основним тілом на поточному шарі. Потребує підтримок.

**Overhang (Нависання)** — частина моделі, яка нависає над порожнечею. Кут більше 45° зазвичай потребує підтримок.

**Hollow (Порожниста модель)** — модель з порожниною всередині для економії смоли. Потребує дренажних отворів.

**Drain Holes (Дренажні отвори)** — отвори в порожнистій моделі для витікання залишкової смоли.

### Післяобробка

**Wash (Миття)** — процес видалення залишкової неполімеризованої смоли з поверхні моделі. Зазвичай використовується ізопропіловий спирт (IPA) або спеціальні розчинники.

**Cure (Полімеризація)** — фінальне висвітлення моделі ультрафіолетом для завершення полімеризації та досягнення максимальної міцності.

**Cure Station (Камера полімеризації)** — пристрій для рівномірного висвітлення моделі з усіх боків.

**Post-Cure (Післяполімеризація)** — процес додаткової полімеризації після миття для підвищення міцності та стабільності.

### Проблеми та явища

**Delamination (Розшарування)** — відрив шарів один від одного. Може бути через недостатню експозицію або проблеми з підтримками.

**Elephant Foot (Слоняча лапа)** — розширення нижніх шарів моделі через занадто великий час експозиції нижніх шарів.

**Layer Lines (Лінії шарів)** — видимі лінії між шарами. У SLA менш помітні, ніж у FDM, але все ж присутні.

**Suction Cup Effect (Ефект присоски)** — коли велика плоска поверхня моделі створює вакуум при підйомі, що може призвести до відриву від платформи.

**Blooming (Розмиття)** — артефакт, коли світло частково проникає в навколишні ділянки, створюючи розмиті краї.

### Матеріали

**Resin (Смола)** — фотополімерна рідина для друку.

**Standard Resin (Стандартна смола)** — базовий тип смоли для загального використання.

**ABS-like Resin (ABS-подібна смола)** — смола з властивостями, схожими на ABS-пластик (міцніша, гнучкіша).

**Tough Resin (Міцна смола)** — смола з підвищеною міцністю та ударною в'язкістю.

**Flexible Resin (Гнучка смола)** — смола, яка після полімеризації залишається гнучкою.

**Water-Washable Resin (Водорозчинна смола)** — смола, яку можна мити водою замість спирту.

**Clear/Transparent Resin (Прозора смола)** — смола, яка після полімеризації стає прозорою або напівпрозорою.

### Програмне забезпечення

**Slicer (Слайсер)** — програма, яка перетворює 3D-модель на інструкції для принтера (наприклад, ChiTuBox, Lychee Slicer).

**STL File (STL-файл)** — найпоширеніший формат 3D-моделей для друку.

**G-code / CTB File** — файл з інструкціями для принтера, створений слайсером.

**Preview (Попередній перегляд)** — можливість побачити, як буде виглядати модель після друку, переглядаючи шари.

**Hollow Tool (Інструмент порожнистості)** — функція в слайсері для створення порожнистих моделей.

**Support Generator (Генератор підтримок)** — автоматичне створення підтримок для моделі.

Ці терміни допоможуть вам краще розуміти процес SLA-друку та спілкуватися з іншими ентузіастами 3D-друку.
