# 4. Фотополімерні смоли

Що охоплює цей розділ:

- **що таке фотополімерна смола** — склад, властивості, поведінка
- **основні типи смол** — стандартні, міцні (ABS-like, Tough), гнучкі, високодеталізовані, спеціальні (стоматологія, ювелірка тощо)

---

## 4.1. Що таке фотополімерна смола

Фотополімерна смола — це спеціальна рідина, яка під дією ультрафіолетового світла перетворюється з рідкого стану в твердий пластик. Це хімічний процес полімеризації, який відбувається миттєво під впливом світла певної довжини хвилі.

### Хімічний склад

Смола складається з кількох основних компонентів:

**Мономери** — основна складова, маленькі молекули, які під дією UV-світла з'єднуються в довгі ланцюги (полімери). Це те, що робить смолу твердою після висвітлення.

**Фотоініціатори** — спеціальні речовини, які поглинають ультрафіолетове світло та запускають реакцію полімеризації. Без них смола не затвердіє навіть під світлом.

**Пігменти** — додають смолі колір. Можуть бути прозорими (clear), кольоровими або навіть з ефектами (металік, перламутр).

**Додаткові добавки** — модифікатори, які змінюють властивості смоли:
- Пластифікатори — для гнучкості
- УФ-стабілізатори — для стійкості до сонячного світла
- Антиоксиданти — для збереження якості під час зберігання

### Фізичні властивості

**В рідкому стані:**
- В'язкість — зазвичай схожа на мед або сироп
- Колір — може бути будь-яким (прозора, кольорова, непрозора)
- Запах — більшість смол мають характерний хімічний запах
- Щільність — трохи важча за воду (близько 1.1–1.2 г/см³)

**В твердому стані (після полімеризації):**
- Твердість — залежить від типу смоли (від м'якої гумоподібної до дуже твердої)
- Міцність — може бути крихкою або міцною, залежить від типу
- Глянець — зазвичай гладка, блискуча поверхня
- Колір — може змінюватися після полімеризації (особливо прозорі смоли)

### Як працює полімеризація

1. **Початковий стан** — смола в рідкому стані, мономери вільно плавають
2. **Вплив UV-світла** — фотоініціатор поглинає світло та активується
3. **Запуск реакції** — активований фотоініціатор починає з'єднувати мономери
4. **Формування полімерів** — мономери з'єднуються в довгі ланцюги
5. **Затвердіння** — смола перетворюється в твердий пластик
6. **Післяполімеризація** — процес продовжується під дією світла або часу

### Важливі особливості

**Чутливість до світла:**
- Смола починає полімеризуватися під будь-яким ультрафіолетовим світлом
- Навіть сонячне світло може почати процес затвердіння
- Тому смолу завжди зберігають в темних пляшках та захищають від світла

**Термін придатності:**
- Невідкрита смола: зазвичай 12–24 місяці
- Відкрита смола: залежить від умов зберігання (3–12 місяців)
- Після закінчення терміну смола може стати густішою або втратити властивості

**Температурна чутливість:**
- Більшість смол працюють оптимально при 20–30°C
- При низькій температурі смола стає густішою, що ускладнює друк
- При високій температурі смола стає рідшою, що може вплинути на якість

**Токсичність:**
- Смоли містять хімічні речовини, які можуть бути шкідливими
- Випари можуть подразнювати дихальні шляхи
- Контакт зі шкірою може викликати подразнення або алергію
- Обов'язкове використання засобів захисту (рукавички, вентиляція)

## 4.2. Основні типи смол

Вибір правильної смоли критично важливий для успішного друку. Різні типи смол мають різні властивості та призначені для різних завдань.

### Стандартна смола (Standard Resin)

**Опис:** Базовий тип смоли, найпоширеніший та найдоступніший.

**Властивості:**
- Середня міцність та твердість
- Гарна деталізація
- Гладка поверхня після друку
- Доступна ціна

**Застосування:**
- Перші друки та навчання
- Декоративні моделі
- Мініатюри та фігурки
- Прототипи з невисокими вимогами до міцності

**Переваги:**
- Низька ціна
- Легко знайти та купити
- Проста в роботі
- Підходить для більшості завдань

**Недоліки:**
- Крихка (легко ламається)
- Не підходить для функціональних деталей
- Може жовтіти під сонячним світлом

**Рекомендації:**
- Ідеальна для початківців
- Добра для тестування налаштувань принтера
- Використовуйте для декоративних об'єктів

### Міцна смола (ABS-like, Tough Resin)

**Опис:** Смоли з підвищеною міцністю та ударною в'язкістю, схожі на ABS-пластик.

**Властивості:**
- Висока міцність на розрив
- Підвищена ударна в'язкість (не ламається від ударів)
- Гнучкість (трохи гнеться перед поломкою)
- Хороша стійкість до зносу

**Застосування:**
- Функціональні прототипи
- Деталі, які будуть використовуватися
- Механічні компоненти
- Заміна деталей для пристроїв
- Іграшки та моделі, які можуть падати

**Переваги:**
- Міцніша за стандартну смолу в 2–3 рази
- Менш крихка
- Підходить для функціональних деталей
- Краща стійкість до механічних навантажень

**Недоліки:**
- Дорожча за стандартну смолу
- Може бути трохи менша деталізація
- Потребує більшого часу експозиції

**Рекомендації:**
- Використовуйте для деталей, які будуть використовуватися
- Ідеальна для механічних прототипів
- Добра альтернатива FDM для функціональних деталей

### Гнучка смола (Flexible / Elastic Resin)

**Опис:** Смоли, які після полімеризації залишаються гнучкими та еластичними, схожі на гумові.

**Властивості:**
- Висока еластичність (може розтягуватися та згинатися)
- М'яка на дотик
- Відновлює форму після деформації
- Різна твердість (від м'якої до середньої)

**Застосування:**
- Шарніри та з'єднання
- Прокладки та уплотнювачі
- М'які іграшки
- Захисні ковпачки
- Еластичні компоненти

**Переваги:**
- Унікальна властивість гнучкості
- Не ламається при згинанні
- Може замінити гумові деталі

**Недоліки:**
- Дорога
- Складніша в роботі (потребує особливих налаштувань)
- Менша міцність на розрив
- Може бути липкою після друку

**Рекомендації:**
- Використовуйте для спеціальних завдань, де потрібна гнучкість
- Потребує досвіду в налаштуванні параметрів
- Може бути складнішою в післяобробці

### Високодеталізована смола (High Detail / 8K / 4K Resin)

**Опис:** Спеціальні смоли, оптимізовані для максимальної деталізації та точності.

**Властивості:**
- Найвища деталізація (може друкувати деталі до 0.01 мм)
- Мінімальна усадка
- Висока точність розмірів
- Гладка поверхня без видимих шарів

**Застосування:**
- Ювелірні вироби
- Мініатюри з дрібними деталями
- Стоматологічні моделі
- Архітектурні макети
- Моделі з тонкими стінками

**Переваги:**
- Найкраща якість поверхні
- Можливість друкувати дуже дрібні деталі
- Мінімальні артефакти друку

**Недоліки:**
- Дорога
- Може бути крихкішою
- Потребує точних налаштувань принтера
- Може бути складнішою в роботі

**Рекомендації:**
- Використовуйте для проектів, де важлива деталізація
- Потребує високої роздільної здатності принтера (4K або 8K)
- Ідеальна для ювелірної справи та мініатюр

### Прозора смола (Clear / Transparent Resin)

**Опис:** Смоли, які після полімеризації стають прозорими або напівпрозорими.

**Властивості:**
- Прозорість (може бути повністю прозорою або напівпрозорою)
- Можливість полірування до кристальної прозорості
- Може бути пофарбована після друку
- Світлопровідність

**Застосування:**
- Оптичні моделі
- Вікна та лінзи (для прототипів)
- Декоративні об'єкти
- Моделі, які потребують прозорості
- Світлодіодні проекти

**Переваги:**
- Унікальна властивість прозорості
- Можливість полірування
- Естетично приваблива

**Недоліки:**
- Може жовтіти з часом
- Складніша в післяобробці (потрібне полірування для повної прозорості)
- Може бути крихкою
- Потребує обережного зберігання

**Рекомендації:**
- Після друку поліруйте для максимальної прозорості
- Захищайте від ультрафіолету, щоб уникнути жовтіння
- Використовуйте для спеціальних проектів

### Водорозчинна смола (Water-Washable Resin)

**Опис:** Смоли, які можна мити водою замість токсичних розчинників.

**Властивості:**
- Миття водою замість IPA
- Безпечніша для навколишнього середовища
- Менше токсичних випарів
- Властивості схожі на стандартну смолу

**Застосування:**
- Для тих, хто хоче уникнути токсичних розчинників
- Друки в домашніх умовах без професійної вентиляції
- Початківці, які хочуть простіший процес

**Переваги:**
- Безпечніша (не потрібен IPA)
- Простіша післяобробка
- Менше витрат на розчинники
- Екологічніша

**Недоліки:**
- Може бути трохи дорожчою
- Менша стійкість до води після полімеризації
- Може потребувати більшого часу експозиції

**Рекомендації:**
- Ідеальна для початківців
- Добра для домашнього використання
- Перевірте сумісність з вашим принтером

### Стоматологічна смола (Dental Resin)

**Опис:** Спеціалізовані смоли для стоматологічних застосувань.

**Властивості:**
- Біосумісність (безпечна для контакту з ротовою порожниною)
- Висока точність
- Спеціальні кольори (білий, рожевий для ясен)
- Відповідає медичним стандартам

**Застосування:**
- Зубні протези та коронки
- Ортодонтичні моделі
- Хірургічні шаблони
- Моделі для планування операцій

**Переваги:**
- Безпечна для медичного використання
- Висока точність
- Спеціалізовані властивості

**Недоліки:**
- Дуже дорога
- Потребує спеціальних налаштувань
- Обмежена доступність

**Рекомендації:**
- Тільки для професійного медичного використання
- Потребує знань медичних стандартів
- Не використовуйте звичайні смоли для медичних цілей

### Ювелірна смола (Jewelry / Castable Resin)

**Опис:** Спеціальні смоли для створення моделей для лиття металів.

**Властивості:**
- Випалюється без залишків (при нагріванні повністю згоряє)
- Висока деталізація
- Точні розміри
- Спеціальна формула для лиття

**Застосування:**
- Моделі для втраченого воску (lost-wax casting)
- Ювелірні вироби
- Лиття з дорогоцінних металів
- Точні копії для лиття

**Переваги:**
- Ідеальна для лиття
- Висока деталізація
- Без залишків після випалювання

**Недоліки:**
- Дорога
- Потребує спеціального обладнання для лиття
- Складніша в роботі

**Рекомендації:**
- Тільки для ювелірної справи
- Потребує знань процесу лиття
- Не використовуйте для звичайних друків

### Керамічна смола (Ceramic Resin)

**Опис:** Смоли, які після спеціальної обробки стають схожими на кераміку.

**Властивості:**
- Після випалювання стає керамікою
- Висока міцність після обробки
- Термостійкість
- Спеціальна обробка

**Застосування:**
- Керамічні вироби
- Термостійкі деталі
- Спеціалізовані проекти

**Переваги:**
- Унікальні властивості кераміки
- Термостійкість

**Недоліки:**
- Дуже дорога
- Потребує спеціальної обробки (випалювання)
- Складніша в роботі

### Кольорові та спеціальні смоли

**Металічні смоли:**
- Імітують метал (золото, срібло, бронза)
- Можуть бути поліровані до металевого блиску
- Використовуються для декоративних цілей

**Флуоресцентні смоли:**
- Світяться під ультрафіолетовим світлом
- Декоративні ефекти
- Спеціальні проекти

**Кольорові смоли:**
- Широкий вибір кольорів
- Можуть бути пофарбовані після друку
- Для декоративних цілей

### Як вибрати правильну смолу

**Для початківців:**
1. Почніть зі стандартної смоли
2. Спробуйте водорозчинну для простішої післяобробки
3. Експериментуйте з різними кольорами

**Для функціональних деталей:**
1. Використовуйте ABS-like або Tough смолу
2. Перевірте вимоги до міцності
3. Розгляньте гнучку смолу для спеціальних випадків

**Для високої деталізації:**
1. Високодеталізована смола
2. Переконайтеся, що принтер підтримує високу роздільність
3. Уважно налаштуйте параметри друку

**Для спеціальних завдань:**
1. Використовуйте спеціалізовані смоли
2. Перевірте сумісність з принтером
3. Дотримуйтеся рекомендацій виробника

### Загальні рекомендації

- **Завжди читайте інструкції** виробника смоли
- **Перевіряйте сумісність** з вашим принтером
- **Експериментуйте** з різними типами для різних проектів
- **Зберігайте смолу правильно** (темне місце, правильна температура)
- **Використовуйте рекомендовані параметри** друку для кожної смоли
- **Не змішуйте різні типи смол** без перевірки сумісності

Правильний вибір смоли — це половина успіху в SLA-друку. Виберіть смолу, яка відповідає вашим потребам та можливостям принтера.
