# 3. Обладнання

Що охоплює цей розділ:

- **з чого складається SLA-принтер** — корпус, LCD/лазер/проектор, ванночка (vat), FEP/ACF плівка, платформа
- **додаткове обладнання** — мийка (wash), камера полімеризації (cure), ультрафіолетове світло, витратні матеріали

---

## 3.1. З чого складається SLA-принтер

SLA-принтер — це складний механізм, який поєднує механічні, оптичні та електронні компоненти. Розуміння конструкції допоможе вам краще обслуговувати принтер та вирішувати проблеми.

### Корпус та рама

**Корпус** — зовнішня оболонка принтера, яка виконує кілька важливих функцій:

- **Захист від світла** — запобігає потраплянню зовнішнього світла до смоли, яка може почати полімеризуватися заздалегідь
- **Захист від пилу** — зберігає внутрішні компоненти чистими
- **Безпека** — закриває ультрафіолетове світло, яке може бути шкідливим для очей
- **Термоізоляція** — допомагає підтримувати стабільну температуру всередині

Більшість настільних принтерів мають корпус з пластику або металу з прозорими вікнами для спостереження за процесом друку.

**Рама** — внутрішня конструкція, яка утримує всі компоненти. Має бути жорсткою, щоб забезпечити точність позиціювання платформи.

### Система висвітлення

Це серце принтера — те, що перетворює рідку смолу на твердий пластик.

#### LCD-екран (для MSLA-принтерів)

**Що це:** Спеціальний монохромний LCD-екран з ультрафіолетовою підсвіткою. Працює як "трафарет" — показує чорно-біле зображення шару, де білі ділянки пропускають світло, а чорні — блокуют його.

**Характеристики:**
- **Роздільна здатність** — визначає мінімальний розмір деталей (наприклад, 4K = 3840×2400 пікселів)
- **Розмір екрану** — визначає максимальний розмір області друку
- **Термін служби** — зазвичай 2000–4000 годин роботи, після чого якість починає погіршуватися

**Догляд:** LCD-екран потребує захисту від смоли. Якщо смола потрапить на екран, його потрібно негайно очистити. Деякі принтери мають захисне скло над екраном.

#### Лазер (для SLA-принтерів)

**Що це:** Ультрафіолетовий лазер, який рухається по поверхні смоли за допомогою гальванометрів (дзеркал), "малюючи" кожен шар точку за точкою.

**Переваги:** Найвища точність та якість поверхні.

**Недоліки:** Повільніший друк, складніше обслуговування, дорожче обладнання.

#### Проектор (для DLP-принтерів)

**Що це:** Цифровий проектор з ультрафіолетовим джерелом світла, який висвітлює весь шар одночасно через систему мікро-дзеркал (DMD).

**Переваги:** Швидкий друк, висока якість, довговічність.

### Ванночка (Vat)

**Ванночка** — це контейнер, який утримує рідку смолу під час друку.

**Конструкція:**
- **Корпус** — зазвичай з алюмінію або пластику, має ручки для зручного зняття
- **Прозоре дно** — через нього проходить ультрафіолетове світло
- **FEP/ACF плівка** — натягнута на дно ванночки, створює розділювальний шар між смолою та екраном

**Розміри:** Визначають максимальний розмір моделі, яку можна надрукувати. Типові розміри: 6.1", 8.9", 10.1" по діагоналі.

**Догляд:** Ванночку потрібно регулярно очищати від залишків смоли та перевіряти стан плівки.

### FEP / ACF плівка

**FEP (Fluorinated Ethylene Propylene)** — найпоширеніша плівка для ванночок.

**Властивості:**
- Прозора для ультрафіолетового світла
- Міцна та стійка до хімічних речовин
- Має низький коефіцієнт тертя, що полегшує відрив моделі
- Термін служби: залежить від інтенсивності використання (зазвичай 20–50 друків)

**ACF (Anti-Creep Film)** — новіша альтернатива FEP.

**Переваги ACF:**
- Менше опору при відриві моделі від плівки
- Менше ризику відриву моделі від платформи
- Довший термін служби
- Менше "хрустіння" та деформації під час друку

**Недоліки ACF:**
- Дорожча за FEP
- Менш поширена, складніше знайти заміну

**Коли міняти плівку:**
- Появилися подряпини або помутніння
- Моделі почали відриватися від платформи
- Плівка стала м'якою або деформованою
- Після потрапляння твердих частинок смоли на плівку

**Як визначити, що плівка потребує заміни:**
- Візуальні подряпини при огляді на світло
- Часті невдалі друки без інших причин
- Помітне "хрустіння" під час відриву моделі

### Платформа для друку (Build Plate)

**Платформа** — це металева (зазвичай алюмінієва) пластина, до якої прилипає перший шар моделі.

**Конструкція:**
- **Плоска поверхня** — має бути ідеально рівною для рівномірного прилипання
- **Кріплення до Z-осі** — з'єднана з механізмом підйому/опускання
- **Текстура поверхні** — деякі платформи мають шорстку поверхню для кращого прилипання

**Калібрування:** Платформа має бути ідеально паралельною до дна ванночки. Неправильна калібрування призводить до невдалих друків.

**Догляд:**
- Регулярно очищати від залишків смоли
- Не пошкоджувати поверхню при знятті моделей
- Перевіряти рівність поверхні

**Розміри:** Зазвичай трохи менші за розмір ванночки, щоб залишався зазор для смоли.

### Механізм Z-осі (підйому/опускання)

**Що це:** Система, яка піднімає та опускає платформу з точністю до мікрон.

**Типи механізмів:**
- **Гвинтова передача (Lead Screw)** — найпоширеніша, забезпечує високу точність
- **Ремінна передача** — рідше, зазвичай в більш дорогих моделях

**Характеристики:**
- **Точність позиціювання** — зазвичай ±0.01 мм
- **Швидкість підйому/опускання** — налаштовується в слайсері
- **Максимальна висота друку** — визначає максимальну висоту моделі

**Догляд:** Потребує регулярного змащування та очищення від пилу.

### Електроніка та управління

**Контролер** — "мозок" принтера, який керує всіма компонентами.

**Дисплей та інтерфейс** — дозволяє керувати принтером, переглядати прогрес друку, налаштовувати параметри.

**Підключення:**
- **USB** — для передачі файлів з комп'ютера
- **SD-карта** — альтернативний спосіб завантаження моделей
- **Wi-Fi** (в деяких моделях) — бездротове керування

### Додаткові компоненти

**Фільтр вентиляції** — деякі принтери мають вбудований фільтр для очищення повітря від випарів смоли.

**Підігрів** — деякі моделі мають підігрів ванночки для підтримки оптимальної температури смоли (зазвичай 25–30°C).

**Датчики:**
- **Датчик рівня смоли** — перевіряє, чи достатньо смоли в ванночці
- **Датчик температури** — контролює температуру всередині принтера

## 3.2. Додаткове обладнання

Для повноцінної роботи з SLA-принтером недостатньо лише самого принтера. Потрібне додаткове обладнання для післяобробки та безпечної роботи.

### Мийка (Wash Station)

**Призначення:** Видалення залишкової неполімеризованої смоли з поверхні моделі після друку.

**Типи мийок:**

**1. Ручна мийка:**
- **Контейнер з розчинником** — простий пластиковий контейнер з ізопропіловим спиртом (IPA)
- **Ультразвукова ванна** — прискорює процес миття завдяки ультразвуковим хвилям
- **Переваги:** Низька вартість, простота
- **Недоліки:** Потрібна ручна робота, менш ефективна

**2. Автоматична мийка:**
- **Wash & Cure Station** — комбінований пристрій для миття та полімеризації
- **Окрема мийка** — спеціалізований пристрій тільки для миття
- **Переваги:** Автоматизація, краща якість очищення, безпечніше
- **Недоліки:** Вища вартість, займає більше місця

**Як працює:**
1. Модель на платформі поміщається в контейнер з розчинником
2. Розчинник розчиняє залишкову смолу на поверхні
3. Модель обертається або перемішується для рівномірного очищення
4. Після миття модель висушується

**Розчинники:**
- **Ізопропіловий спирт (IPA)** — найпоширеніший, ефективний, але легкозаймистий
- **Вода** — тільки для водорозчинних смол
- **Спеціальні розчинники** — для певних типів смол

**Безпека:** Розчинники токсичні та легкозаймисті. Робота в провітрюваному приміщенні, використання рукавичок.

### Камера полімеризації (Cure Station)

**Призначення:** Фінальна полімеризація моделі після миття для досягнення максимальної міцності та стабільності.

**Типи:**

**1. Прості рішення:**
- **UV-лампа** — ультрафіолетова лампа для ручного висвітлення моделі
- **Сонячне світло** — природне ультрафіолетове світло (повільніше, менш передбачувано)
- **Переваги:** Низька вартість
- **Недоліки:** Неравномірне висвітлення, потрібна ручна робота

**2. Професійні рішення:**
- **Cure Station** — автоматична камера з UV-світлодіодами та обертовою платформою
- **Wash & Cure Station** — комбінований пристрій
- **Переваги:** Рівномірне висвітлення з усіх боків, автоматизація, контрольований час
- **Недоліки:** Вища вартість

**Як працює:**
1. Модель поміщається на обертову платформу
2. Увімкнення UV-світлодіодів з усіх боків
3. Обертання платформи забезпечує рівномірне висвітлення
4. Процес триває 5–15 хвилин залежно від розміру моделі

**Параметри полімеризації:**
- **Час** — залежить від розміру та типу смоли (зазвичай 5–15 хвилин)
- **Температура** — деякі камери мають підігрів для кращої полімеризації
- **Інтенсивність світла** — вища інтенсивність = швидша полімеризація

### Ультрафіолетове світло

**Джерела UV-світла:**

**1. UV-світлодіоди (LED):**
- Найпоширеніші в сучасних принтерах та камерах полімеризації
- Довговічні, енергоефективні
- Довжина хвилі: зазвичай 405 нм (для більшості смол)

**2. UV-лампы:**
- Використовуються в простіших рішеннях
- Можуть бути різних типів (ртутні, флуоресцентні)
- Потребують обережності (можуть містити ртуть)

**3. Сонячне світло:**
- Найдешевше рішення
- Містить ультрафіолетові хвилі
- Ненадійне (залежить від погоди, часу доби)

**Безпека:**
- UV-світло шкідливе для очей — ніколи не дивіться безпосередньо на джерело
- Використовуйте захист очей (окуляри або щиток)
- Захищайте шкіру від тривалого впливу

### Витратні матеріали

**Смола:**
- Основна витрата — фотополімерна смола
- Витрати залежать від об'єму друку та типу смоли
- Рекомендація: мати запас 1–2 пляшки для безперебійної роботи

**FEP/ACF плівка:**
- Потрібна заміна кожні 20–50 друків (залежить від інтенсивності використання)
- Рекомендація: мати запасну плівку завжди під рукою

**Розчинники для миття:**
- Ізопропіловий спирт (IPA) — можна фільтрувати та використовувати повторно
- Рекомендація: мати запас 2–5 літрів

**Рукавички:**
- Нітрилові або латексні рукавички — обов'язкові для роботи зі смолою
- Одноразові, потребують регулярної заміни
- Рекомендація: мати запас упаковки

**Фільтри:**
- Фільтри для очищення використаної смоли
- Фільтри для очищення розчинників
- Рекомендація: мати набір фільтрів різної крупності

**Інструменти:**
- **Скребок (Spatula)** — для зняття моделей з платформи
- **Пінцет** — для роботи з дрібними деталями
- **Ножиці** — для зрізання підтримок
- **Наждачний папір** — для фінішної обробки
- **Мікрофібра** — для очищення компонентів

**Захисні матеріали:**
- **Окуляри захисту** — для захисту очей від UV-світла та смоли
- **Фартух або спецодяг** — для захисту одягу від смоли
- **Підкладка на стіл** — для захисту робочої поверхні

### Організація робочого місця

**Рекомендована схема робочого місця:**

1. **Зона друку** — принтер, джерело живлення
2. **Зона післяобробки** — мийка, камера полімеризації
3. **Зона зберігання** — смоли, витратних матеріалів, інструментів
4. **Зона утилізації** — контейнери для відходів смоли та розчинників

**Вимоги до приміщення:**
- **Вентиляція** — обов'язкова для видалення випарів смоли
- **Температура** — стабільна температура 20–25°C (залежить від типу смоли)
- **Вологість** — низька вологість (висока може впливати на якість друку)
- **Захист від світла** — приміщення не повинно мати прямого сонячного світла
- **Доступ до води** — для миття рук та очищення

**Безпека:**
- Вогнегасник (розчинники легкозаймисті)
- Аптечка першої допомоги
- Інструкції з безпеки на видимій ділянці

### Рекомендації щодо покупки

**Мінімальний набір для старту:**
- SLA-принтер
- 1–2 пляшки смоли
- Контейнер для миття + IPA
- UV-лампа або доступ до сонячного світла
- Рукавички, окуляри, скребок

**Оптимальний набір:**
- Все з мінімального набору +
- Wash & Cure Station (комбінований)
- Запасна FEP/ACF плівка
- Додаткові інструменти (пінцет, ножиці)
- Ультразвукова ванна (опціонально)

**Професійний набір:**
- Все з оптимального набору +
- Додаткові ванночки для різних смол
- Система фільтрації розчинників
- Контрольоване середовище (температура, вологість)

Правильний вибір обладнання залежить від ваших потреб, бюджету та планів щодо використання принтера. Почніть з мінімального набору та додавайте обладнання за потреби.
